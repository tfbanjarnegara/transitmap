# Peta Jaringan Angkutan Umum Kabupaten Banjarnegara
### *Banjarnegara Public Transportation Network Map*

Peta ini meliputi 20 kecamatan di Kabupaten Banjarnegara, mencakup bus medium antar kota, mikro bus, dan angkot lokal.
*This map covers 20 districts in Banjarnegara Regency, including intercity medium buses, micro-buses, and local angkot services.*

### ⚠️ Informasi Penting / *Essential Information*

* **Sistem Naik-Turun Bebas (Stop-on-Request):**
  Titik pemberhentian adalah rujukan/referensi, bukan titik berhenti resmi. 
  *Stops shown are reference points only; the service operates on a stop-on-request basis.*

* **Metode Pembayaran / *Payment Method*:**
  Hanya menerima **Tunai**. Tidak melayani QRIS atau kartu (BCA Flazz/TapCash/Brizzi).
  *Cash only. Electronic payments (QRIS/E-money cards) are not accepted.*
